# MovieTime
Aplikasi MovieTime.
Aplikasi untuk mengetahui tentang movie maupun aktor pemerannya

# Get Started
1. Ketik 'git clone https://github.com/ilhamfadilahh/MovieTime.git' kemudian enter.
2. Jalankan project dengan Flutter v3.x.x keatas
